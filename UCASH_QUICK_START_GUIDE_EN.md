# UCASH - Quick Start Guide (English)

## Getting Started with UCASH

### System Overview
UCASH is a comprehensive money transfer and financial management application supporting USD and CDF currencies with real-time synchronization.

---

## Administrator Quick Start

### First Login
1. **Launch UCASH application**
2. **Login with default credentials:**
   - Username: `admin`
   - Password: `admin123`
3. **Select language:** English or French
4. **Wait for initial synchronization**

### Essential First Steps
1. **Create Your First Shop:**
   - Go to **Shops** → **Add Shop**
   - Enter shop details (name, location, initial capital)
   - Save shop information

2. **Add Your First Agent:**
   - Navigate to **Agents** → **Create Agent**
   - Fill personal information
   - Assign to shop
   - Generate automatic matricule (format: AGT[YY][MM][DD][ShopCode][XXX])
   - Set login credentials

3. **Configure Exchange Rates:**
   - Go to **Rates and Commissions**
   - Set USD/CDF exchange rates
   - Configure transaction fees

4. **Set Up Personnel (Optional):**
   - Navigate to **Personnel**
   - Add employees with salary information
   - Configure payroll settings

### Daily Operations
- **Monitor Dashboard:** Check system statistics and recent activities
- **Review Reports:** Access financial and operational reports
- **Handle Validations:** Process deletion requests and admin validations
- **Manage Synchronization:** Monitor sync status across all devices

## Agent Quick Start

### First Login
1. **Get credentials from administrator**
2. **Launch UCASH application**
3. **Select Agent login**
4. **Enter provided username and password**
5. **Select preferred language**

### Daily Workflow

#### Morning Setup
1. **Check Dashboard:** Review overnight activities and notifications
2. **Verify Cash Balance:** Confirm opening cash balance
3. **Review Pending Validations:** Check items requiring attention
4. **Sync Data:** Ensure latest data is synchronized

#### Processing Transactions

**For Deposits:**
1. Go to **Operations** → **Deposits**
2. Enter client information
3. Specify amount and currency
4. Process transaction
5. Provide receipt to client

**For Withdrawals:**
1. Go to **Operations** → **Withdrawals**
2. Verify client identity
3. Check available balance
4. Process withdrawal
5. Count and provide cash

**For Virtual Transactions:**
1. Navigate to **VIRTUEL**
2. Select appropriate SIM card
3. Process mobile money transaction
4. Update virtual balances
5. Handle cash reconciliation

#### End of Day
1. **Reconcile Cash:** Balance physical cash with system records
2. **Generate Reports:** Create daily transaction summaries
3. **Process Closures:** Complete daily closure procedures
4. **Sync Data:** Ensure all transactions are synchronized

### Key Features to Master
- **Multi-Currency Operations:** Handle both USD and CDF transactions
- **Virtual Transaction Management:** Process mobile money operations
- **Float Management:** Request and manage cash float
- **Validation Processing:** Handle pending transaction validations

## Common Tasks Reference

### Multi-Currency Operations
- **USD Transactions:** Primary currency for cash operations
- **CDF Transactions:** Secondary currency for virtual operations
- **Exchange Rates:** Automatically applied based on current rates
- **Currency Display:** 
  - Cash amounts: Always "Cash USD"
  - Virtual amounts: Show original currency (USD/CDF)

### Synchronization
- **Automatic Sync:** Occurs every few minutes
- **Manual Sync:** Use sync button when needed
- **Offline Mode:** Continue working without internet
- **Conflict Resolution:** System handles data conflicts automatically

### Reporting
- **Daily Reports:** Transaction summaries and cash flow
- **Monthly Reports:** Performance and revenue analysis
- **Custom Reports:** Filter by date, currency, transaction type
- **Export Options:** PDF, Excel, CSV formats

### Security Best Practices
- **Strong Passwords:** Use complex passwords
- **Regular Logout:** Log out when away from workstation
- **Verify Identity:** Always confirm client identity
- **Report Issues:** Immediately report suspicious activities

## Troubleshooting Quick Fixes

### Login Issues
- **Check credentials:** Verify username and password
- **Check internet:** Ensure stable connection
- **Clear cache:** Reset application data
- **Contact admin:** If problem persists

### Sync Problems
- **Manual sync:** Force synchronization
- **Check connection:** Verify internet connectivity
- **Restart app:** Close and reopen application
- **Wait and retry:** Server may be temporarily unavailable

### Transaction Errors
- **Check balance:** Verify sufficient funds
- **Verify details:** Confirm transaction information
- **Retry operation:** Attempt transaction again
- **Contact support:** Seek assistance if needed
