# UCASH - Guide de Démarrage Rapide (Français)

## Commencer avec UCASH

### Vue d'Ensemble du Système
UCASH est une application complète de transfert d'argent et de gestion financière supportant les devises USD et CDF avec synchronisation en temps réel.

---

## Démarrage Rapide Administrateur

### Première Connexion
1. **Lancer l'application UCASH**
2. **Se connecter avec les identifiants par défaut :**
   - Nom d'utilisateur : `admin`
   - Mot de passe : `admin123`
3. **Sélectionner la langue :** Anglais ou Français
4. **Attendre la synchronisation initiale**

### Étapes Essentielles
1. **Créer Votre Premier Shop :**
   - Aller à **Shops** → **Ajouter Shop**
   - Saisir les détails du shop (nom, localisation, capital initial)
   - Sauvegarder les informations

2. **Ajouter Votre Premier Agent :**
   - Naviguer vers **Agents** → **Créer Agent**
   - Remplir les informations personnelles
   - Assigner au shop
   - Générer le matricule automatique (format : AGT[AA][MM][JJ][CodeShop][XXX])
   - Définir les identifiants de connexion

3. **Configurer les Taux de Change :**
   - Aller à **Taux et Commissions**
   - Définir les taux de change USD/CDF
   - Configurer les frais de transaction

4. **Configurer le Personnel (Optionnel) :**
   - Naviguer vers **Personnel**
   - Ajouter des employés avec informations salariales
   - Configurer les paramètres de paie

### Opérations Quotidiennes
- **Surveiller le Tableau de Bord :** Vérifier les statistiques système et activités récentes
- **Réviser les Rapports :** Accéder aux rapports financiers et opérationnels
- **Traiter les Validations :** Traiter les demandes de suppression et validations admin
- **Gérer la Synchronisation :** Surveiller le statut de sync sur tous les appareils

## Démarrage Rapide Agent

### Première Connexion
1. **Obtenir les identifiants de l'administrateur**
2. **Lancer l'application UCASH**
3. **Sélectionner connexion Agent**
4. **Saisir le nom d'utilisateur et mot de passe fournis**
5. **Sélectionner la langue préférée**

### Flux de Travail Quotidien

#### Configuration Matinale
1. **Vérifier le Tableau de Bord :** Réviser les activités nocturnes et notifications
2. **Vérifier le Solde Cash :** Confirmer le solde d'ouverture
3. **Réviser les Validations en Attente :** Vérifier les éléments nécessitant attention
4. **Synchroniser les Données :** S'assurer que les dernières données sont synchronisées

#### Traitement des Transactions

**Pour les Dépôts :**
1. Aller à **Opérations** → **Dépôts**
2. Saisir les informations client
3. Spécifier le montant et la devise
4. Traiter la transaction
5. Fournir le reçu au client

**Pour les Retraits :**
1. Aller à **Opérations** → **Retraits**
2. Vérifier l'identité du client
3. Vérifier le solde disponible
4. Traiter le retrait
5. Compter et fournir l'argent

**Pour les Transactions Virtuelles :**
1. Naviguer vers **VIRTUEL**
2. Sélectionner la carte SIM appropriée
3. Traiter la transaction mobile money
4. Mettre à jour les soldes virtuels
5. Gérer la réconciliation cash

#### Fin de Journée
1. **Réconcilier le Cash :** Équilibrer l'argent physique avec les enregistrements système
2. **Générer les Rapports :** Créer les résumés de transactions quotidiennes
3. **Traiter les Clôtures :** Compléter les procédures de clôture quotidienne
4. **Synchroniser les Données :** S'assurer que toutes les transactions sont synchronisées

### Fonctionnalités Clés à Maîtriser
- **Opérations Multi-Devises :** Gérer les transactions USD et CDF
- **Gestion des Transactions Virtuelles :** Traiter les opérations mobile money
- **Gestion des Flottants :** Demander et gérer le flottant cash
- **Traitement des Validations :** Gérer les validations de transactions en attente

## Démarrage Rapide Client

### Accès au Compte
1. **Obtenir les identifiants de votre agent**
2. **Lancer l'application UCASH**
3. **Sélectionner connexion Client**
4. **Saisir nom d'utilisateur et mot de passe**
5. **Choisir la langue préférée**

### Services Disponibles

#### Voir les Informations du Compte
- **Tableau de Bord :** Voir le solde actuel et transactions récentes
- **Historique des Transactions :** Réviser l'historique complet des transactions
- **Détails du Compte :** Voir les informations personnelles et statut du compte

#### Demander des Services
1. **Sélectionner le service désiré** (transfert, retrait, dépôt)
2. **Remplir les détails de transaction**
3. **Soumettre la demande**
4. **Attendre le traitement par l'agent**
5. **Recevoir la confirmation**

#### Gestion du Compte
- **Mettre à Jour les Informations de Contact :** Modifier téléphone, email, adresse
- **Changer le Mot de Passe :** Mettre à jour les identifiants de connexion
- **Voir les Relevés :** Accéder aux relevés de compte et rapports

## Référence des Tâches Communes

### Opérations Multi-Devises
- **Transactions USD :** Devise principale pour les opérations cash
- **Transactions CDF :** Devise secondaire pour les opérations virtuelles
- **Taux de Change :** Appliqués automatiquement selon les taux actuels
- **Affichage des Devises :**
  - Montants cash : Toujours "Cash USD"
  - Montants virtuels : Afficher la devise originale (USD/CDF)

### Synchronisation
- **Sync Automatique :** Se produit toutes les quelques minutes
- **Sync Manuelle :** Utiliser le bouton sync quand nécessaire
- **Mode Hors Ligne :** Continuer à travailler sans internet
- **Résolution de Conflits :** Le système gère automatiquement les conflits de données

### Rapports
- **Rapports Quotidiens :** Résumés de transactions et flux de trésorerie
- **Rapports Mensuels :** Analyse de performance et revenus
- **Rapports Personnalisés :** Filtrer par date, devise, type de transaction
- **Options d'Export :** Formats PDF, Excel, CSV

### Bonnes Pratiques de Sécurité
- **Mots de Passe Forts :** Utiliser des mots de passe complexes
- **Déconnexion Régulière :** Se déconnecter quand absent du poste
- **Vérifier l'Identité :** Toujours confirmer l'identité du client
- **Signaler les Problèmes :** Signaler immédiatement les activités suspectes
